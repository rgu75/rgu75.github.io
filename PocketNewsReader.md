# Pocket News Reader

Technically, this application is a RSS and Atom feed aggregator. If that does not ring a bell, don't worry. Pocket News allows you, in a single place, to track additions and changes in your favourite websites.

Special feature: autodiscovery
Using the integrated web browser, navigate to your site (usually on the main page, or a page that references feeds) and let Pocket News find the links for you.

Happy reading!

## Privacy Policy

Pocket News does not collect, nor use any personal information. All application data and settings are stored in the private user local storage on the device, and are not accessed by any third-party application.
